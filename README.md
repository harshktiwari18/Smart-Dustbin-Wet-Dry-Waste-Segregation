# Smart Dustbin for Automatic Wet and Dry Waste Segregation

## Overview
This project is an Arduino Nano-based smart dustbin that automatically segregates wet and dry waste using an HC-SR04 Ultrasonic Sensor, HW-028 Rain Drop Sensor, and SG90 Servo Motor. It reduces manual effort and promotes efficient waste management through automation.

## Features
- Automatic wet and dry waste segregation
- Contactless waste detection
- Moisture-based waste classification
- Servo motor controlled waste routing
- Low-cost and energy-efficient
- Easy to build and maintain

## Hardware Components
- Arduino Nano
- HC-SR04 Ultrasonic Sensor
- HW-028 Rain Drop Sensor
- SG90 Servo Motor
- Breadboard
- Jumper Wires
- 9V Battery
- Cardboard Prototype

## Software Used
- Arduino IDE
- Embedded C (Arduino C/C++)

## Working Principle
1. Ultrasonic sensor detects the waste.
2. Rain sensor checks whether the waste is wet or dry.
3. Arduino Nano processes the sensor data.
4. Servo motor rotates towards the appropriate compartment.
5. Servo returns to the default position after waste disposal.
